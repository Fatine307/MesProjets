# Projet Excel/VBA – Outil d’analyse budgétaire

## Description (FR)
Développement d’un tableau de bord automatisé sous Excel avec VBA.  
Objectif : faciliter l’analyse, la planification et la prise de décision financière à l’aide d’outils interactifs et probabilistes.

## Description (EN)
Development of an automated Excel dashboard with VBA.  
Objective: facilitate financial analysis, planning, and decision-making using interactive and probabilistic tools.

## Fonctionnalités / Features
- Centralisation et visualisation des dépenses / Centralization and visualization of expenses  
- Optimisation budgétaire avec Solver / Budget optimization with Solver  
- Simulations stochastiques (Monte Carlo) / Stochastic simulations (Monte Carlo)  
- Tableaux dynamiques et visualisations interactives / Pivot tables and interactive visualizations  

## Contenu / Content
- `Projet FINAL.xlsm` : fichier Excel complet avec macros / full Excel file with macros  
- `README.md` : explications et instructions / explanations and instructions  

## Prérequis / Requirements
-  **Avertissement : certaines fonctionnalités du projet ne sont pas prises en charge sur macOS. L’utilisation d’un ordinateur fonctionnant sous Windows avec Microsoft Excel est fortement recommandée. / Warning : Some features of the project are not supported on macOS. It is strongly recommended to use a computer running Windows with Microsoft Excel.**
- Microsoft Excel avec support VBA  
- Activer les macros / Enable macros
